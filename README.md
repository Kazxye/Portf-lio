# Kazys - Portfolio

Meu portfólio pessoal desenvolvido com React, TypeScript e Tailwind CSS.

## Tecnologias

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Vite

## Como rodar

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build
```
